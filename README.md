# Pizza Delivery System

## Course
CIS-18A Java Programming

## Description
This program allows customers to place a pizza order, select size and toppings, choose delivery time, and generate an order summary saved to a text file.

## Features
- Menu selection using switch statements
- Loops for topping selection
- OOP design with multiple classes
- Inheritance and interfaces
- File output for order records
- Input validation

## How to Run
Compile:
javac Main.java

Run:
java Main
